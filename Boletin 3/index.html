Boletín 02

UD1. Introdución a Javascript

Ejercicio 1

Realiza un ejercicio en JavaScript que cuente el número de cifras de un número dado por pantalla

Ejercicio 2

Realiza un ejercicio en JavaScript que pinte por pantalla un triángulo equilátero con n puntos de cada lado. Estos N puntos será el tamaño del triángulo y debe ser un elemento indicado por el usuario.

Ejercicio 3

Realiza un programa que muestre un menú con cuatro opciones. El menú debe mostrarse hasta que se seleccione salir. Se irán almacenando valores y cuando indiquemos el tipo de operación a realizar el programa en ese momento debe mostrarnos el resultados y seguir funcionando.

  1. a. Introducir nuevo número
  2. b. Calcular máximo
  3. c. Calcular mínimo
  4. d. Calcular media
  5. e. Salir

Simulacro examen

Realiza una funcion generadora de contraseñas que reciba por parametros un valor booleano que almacena si la contraseña será fuerte o no fuertre. Para ello, lo primero que haremos es pedir al usuario el numero de caracteres que va a tenar nuestra contraseña. Esto marcará la extensión de la misma. Dependiendo de si será debil(solo una palabra) o fuerte(palabra, numero y caracteres extraños) generará nuestra contraseña. AL finalizar debemos devolver la contraseña generada. Suponed, que los vectores globales con caracteres y palabras están definidos de la siguiente forma: palabras="abcdefghijklmnopqrstuvwxyz"; caracteres="1234567890()/=*#?";

Ejercicio 4

Crea una función en JavaScript que de forma aleatoria calcule una nota de clase y la muestra por pantalla. Además del valor numérico debe mostrar si se trata de Suspenso (0-4), Suficiente (5), Bien (6), Notable(7-8), Sobresaliente(9-10)

Ejercicio 5

Crea un programa que transforme de binario a decimal un número dado por parámetro
